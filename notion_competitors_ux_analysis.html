UX wins — successful design decisions

Product
Notable UX wins
Notion
Slash command: "/" anywhere opens block picker — reduces discoverability friction without toolbars
Inline database: Embed a database inside a page — collapses the gap between docs and data
Template gallery: Community-driven gallery shortens time-to-value for new users significantly
ClickUp
Onboarding checklist: Progress sidebar reduces blank-screen paralysis for new users
Multiple views: Toggle between list, board, calendar, Gantt on the same data — no context switching
Automations: Trigger/action builder is the most comprehensive in this class
Confluence
Jira link: Bidirectional sync between docs and tickets is uniquely valuable for dev teams
Inline comments: Highlight text → leave a comment — reduces review friction significantly
Page analytics: View count and engagement metrics help teams identify stale docs
Coda
Survey-adaptive onboarding: Use-case survey shapes the first experience — higher relevance from step one
Doc Gallery: One of the richest template communities — peer learning accelerates adoption
Formula engine: Spreadsheet-grade formulas in a document context — powerful for ops teams

UX challenges reported across the internet

Notion
Blank canvas paralysis: New users don't know where to start — too much freedom without structure
Database performance: Large databases with many views slow significantly on web
Mobile degraded: Mobile editing experience significantly worse than desktop
ClickUp
UI density overload: 1,100+ G2 mentions of non-intuitive UI; small fonts, competing visual hierarchy
Automation lag: Automations can delay minutes — no clear status feedback during the wait
Dashboard overload: Default view presents too many simultaneous widgets; no progressive disclosure
Confluence
Dated editor UX: Formatting is restrictive and inconsistent — copying content loses structure
Search irrelevance: Search results not ordered by relevance; indexing delays in large instances
Permission complexity: Space-level and page-level permissions conflict — accidental overexposure common
Coda
Power-user curve: 2-week learning period reported — the formula model assumes technical confidence
Large doc lag: Performance degrades noticeably with documents exceeding 10+ pages or large tables
Team adoption friction: Hard to convince non-technical teammates — no familiar mental model
